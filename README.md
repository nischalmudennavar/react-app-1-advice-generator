# Welcome to the Advice App 🧙🏻‍♂️

## This is a simple app that gives you advice 👼🏻

## How to use

1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open your browser and go to `localhost:3000`

## How to contribute

1. Fork the repo
2. Create a new branch
3. Make your changes
4. Create a pull request
